# Personal Finance Manager

A modern personal finance management application built with **Streamlit** and **SQLite**, designed to help users track income and expenses, categorize transactions, analyze financial behavior, and understand the accounting equation in a simple and interactive way.

## Features

* 📊 Interactive financial dashboard
* 💰 Income and expense tracking
* 🗂️ Transaction categorization
* 📈 Financial analysis and spending insights
* 🧮 Accounting equation calculation and visualization
* 🔍 Search and filter transactions
* 💾 Local data storage using SQLite
* 🎨 Clean and user-friendly interface powered by Streamlit

## Technologies Used

* **Python**
* **Streamlit**
* **SQLite**
* **Pandas**
* **Plotly / Matplotlib** (if used)

## Accounting Equation

The application provides a practical implementation of the fundamental accounting equation:

Assets = Liabilities + Equity

Users can visualize how their financial transactions impact each component of the equation in real time.
